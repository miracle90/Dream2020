## SQL

### SQL分类

* DML => Data manipulate Language 数据操作语言（增删改查）
* DDL => Data Define Language 数据定义语言，定义数据库和标的数据结构
* DCL => Data Control Language 数据库控制语言，是用来设置或更改数据库用户或角色权限的语句，包括（grant,revoke等）语句
